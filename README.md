# FFX_Mix_Compendium
This is a companion app for Final Fantasy X item mixing. It breaks down the mixing process into three tabs:  Mixer: Input two items; get an instant, filtered result.  Mix Result Index: Browse all 64 unique Overdrives and their full recipes.  Item Sources: Find exact steal/bribe/drop etc. info for any item. Click any item or mix to jump to its info.
